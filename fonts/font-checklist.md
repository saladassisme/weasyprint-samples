# 字体文件清单

## 当前配置的字体文件

请在 `fonts/` 目录下放置以下字体文件：

### 1. 标题字体（书法风格）
- **文件名**: `custom-title.ttf`
- **用途**: 章节标题、封面标题
- **建议**: 选择具有书法或手写风格的字体
- **示例**: 华文行楷、方正舒体、汉仪行楷等

### 2. 正文字体（宋体风格）
- **文件名**: `custom-body.ttf`
- **用途**: 文章正文段落
- **建议**: 选择清晰易读的宋体或衬线字体
- **示例**: 思源宋体、方正书宋、华文宋体等

### 3. 楷体字体
- **文件名**: `custom-kai.ttf`
- **用途**: 目录、作者信息、其他辅助文本
- **建议**: 选择古典优雅的楷体字体
- **示例**: 华文楷体、方正楷体、汉仪楷体等

## 字体文件检查清单

- [ ] 字体文件已放置在 `fonts/` 目录
- [ ] 文件名与CSS定义一致
- [ ] 字体文件格式正确 (TTF/OTF)
- [ ] 字体文件未损坏
- [ ] 有使用该字体的合法授权

## 测试方法

1. 放置字体文件后运行：
   ```bash
   python generate_book_style.py
   ```

2. 检查生成的PDF是否使用了自定义字体

3. 如果字体未生效，检查：
   - 文件路径是否正确
   - 字体文件是否完整
   - CSS语法是否正确
